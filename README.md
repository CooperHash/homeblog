## 技术栈
```
vue cli4 + vue Router + vuex + webpack + express
```

## 页面
1. 首页动态
2. 添加文章
3. 文章列表展示
4. 文章展示
5. 书籍展示
6. 书籍书评展示
7. 语言展示

## 功能及布局
1. 文章首页引入 lodash 进行文章检索的防抖
2. 语言板块用flex wrap进行宫格布局
3. 日语文章内的音频使用blob链接转化
4. 用伪类及css绑定attr进行动态数组绑定（在不改动业务逻辑的情况下，添加移动端的书籍简介展示）
